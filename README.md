# device-management-frontend
  

## 启动方式
页面由jsp转成HTML为了实现页面和API接口分离，把这些页面文件由nginx托管即可访问。
注：在idea或者webstorm中由于静态资源文件路径引用的问题是不能正常显示的。

## 功能：
### 管理员用户
用户信息的查看与修改；
密码修改；
添加删除用户；
### 企业用户
信息查看与修改；
设备管理；
维修工单管理；
反馈管理；
### 客户
信息管理；
密码修改；
查看设备供应商；
提交维修工单；
反馈；
